# pi-caveman

Caveman mode extension for [pi](https://github.com/mariozechner/pi-coding-agent).

Forces ultra-terse LLM responses: no filler, no preamble, no fluff. Technical substance only.

## Install

```
pi install npm:pi-caveman
```

## Usage

Toggle on/off with `/caveman`.

When active, every LLM response is injected with the Caveman Ultra system prompt — fragments, abbreviations, symbols over words, no tables, no prose paragraphs.

## Uninstall

```
pi remove npm:pi-caveman
```
